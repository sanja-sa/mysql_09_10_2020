База данных ld будет упрощенным аналогом БД по проекту LinkendIn ( социальная сеть для поиска и установления деловых контактов ).
Решаемые задачи:
  1. Хранение и выборка данных работников ( ФИО, профессиональные навыки, резюме, список организаций где работал, текущая работа, должность ... )
  2. Хранение и выборка данных по организациям ( название, расположение, описание ... )
  3. Хранение и выборка информации принадлежности работников к организациям
  4. Хранение и выборка переписки между работниками
  5. Хранение и выборка контактов между работниками
  6. Хранение и выборка данных по вакансиям от организаций
